== HOW TO GET THIS UP AND RUNNING

* Ruby/Rails version: 2.2.2 / 4.2.5 (rvm is recommended to manage this versioning)

* System dependencies: mongodb (can be installed using Homebrew under Mac by using command 'brew install mongodb')

* Database initialization: run 'mongod run --config /usr/local/etc/mongod.conf &'

* How to run it: issue command 'rails server' in the project directory, and then visit http://localhost:3000 using the browser.
